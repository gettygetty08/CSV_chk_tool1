# CSV_chk_tool

# 目的
* C#の勉強とGithubを使ってみて覚えていく
* 何か簡単なものを作ってみる
* csvファイルの取り込み、データのチェック、csvファイルの出力 

# 概要
* csvファイルの取り込み

* 取り込んだデータの異常値チェック

* 異常データがある場合は対象のデータがわかるファイルを出力

  * 異常データを修正して再度CSVファイルを取り込む

* 取り込んだデータのチェック

  * チェックOKデータとNGデータを出力する
